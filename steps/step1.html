Setup Github pages

The first thing we are going to do is to steup an actual webpage so that we have something to test later in our Lambdatest environment. To set up this simple website, we are going to use Github pages but you can ofcourse use other hosting platforms aswell, the important thing is that you can reach your website via an url in a browser.

To get started with Github pages, first, you need to create a repository that we will use to publish to your website. Go to your Github profile and create a new repository. You will need to name the repository username.github.io and make sure username exactly matches your username, else it won’t work. When this is done, head over to your terminal and clone your repository.

This repo will now automatically publish an index.html-file when you push to the master branch, to the URL https://username.github.io. To see this in action, let’s create a simple index.html file with some basic HTML structure to it and push it.
